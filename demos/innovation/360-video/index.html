Getting started

  1. You must already be using JW Player with the Premium Edition license, or get it here.
  2. Follow our JW Player developer documentation for setting up your JW Player.
  3. Download the JW Player HTML5 VR Plugin JS file and host it on your web server.
  4. Add the following code to your JW Player setup configuration block to enable the VR plugin: 

plugins: {
	'https://yourwebserver.com/vr.js': {}
}
